# Onboarding

## Todo:
1. React
- install [nodejs](https://nodejs.org/en/)
- `node -v`
- `npm -v`
2. Clone project
- `git clone https://github.com/Aurorae-Cerebrum/web.git`
- `cd web`
3. Tailwind
- `npm install -D tailwindcss postcss autoprefixer`
- `npx tailwindcss init`

## Run App:
- `cd web`
- `npm run start`
- [view in browser](http://localhost:3000/)
